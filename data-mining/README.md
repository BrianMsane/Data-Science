# Data Mining

There are many definitions of data mining but let's focus on these three for now.

1. Data mining is the process of analyzing data to find pattern, discover trends, and gain insights into how data can be used.
2. Data mining is the process used by companies to turn raw data into useful information.
3. Data mining is the process of discovering intersting patterns and knowledge from large amounts of data.

Note: There are repeating statements in these three definitions of data mining. These statements make-up the core of data mining and they include:

- Data mining is a process.
- Finds patterns, or trends, or insights.
- It works on large amounts of raw data.
- Converts this raw data into either knowledge or information.

## Kinds of Data that can be Mined

Data mining can be applied to any kind of data as long as the data are meaningful to the target application. Various kinds of data include:

- Database Data \
  A database system consists of a collection or interrelated data, known as the database, and a set of software programs used to manage and access data, known as the database management system (DBMS). A common databaset type is called a relational database which is a collection of tables each of which is assigned a unique name. Each table consists of a st of attributes/columns/fields and usually stores a large set of tuples/records defined by a uniqued identifier and described by a set of attribute values. Relational dadta can be accessed by database queries written in a relational query language such as SQL.

  When mining relational databases, we can go further by searching trends and patterns. For example, data mining system can analyze customer data to predict credit risk of new customers based on columns such as age, income, and previous credit information. Data mining system may also detect deviation, that is, items that differ from most of the others. Such deviation can further be investigated.

  Relational databases are the most common available and richest information repositories in organization thus they are a major data form in the study of data mining.

## Kinds or Patterns that can be Mined

There are a number of data mining functionalities and thes include: (1) characterization and discrimination, (2) mining frequent itemsets, associations, and correlations, (3) classification and regression, and (4) clustering analysis and outlier analysis. Data mining functionalities are used to specify the kinds of patterns to be found in data mining tasks. In general, such takss can be classified into two categories whicha re descriptive and predictive.

Descriptive mining tasks characterize properties of data in a target dataset. Predictive mining tasks perform induction on the current data in order to make predictions.

### Class/Concept Description: Characterization and Discrimination

Here, data entries are associated with classes or concepts. These concepts are defined in a summarized, concise, and precise terms, for example, concepts of customers could be big_spenders and budget_spenders. These descriptios are derived using:

- data characterization: summarizing the data of the class under study in general terms
- data discrimination: comparison of the target class with one or a set of comparative or contrasting classes
- both

Data characterization is a summarization of the general characteristics or features of the target class of data
